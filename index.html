About

介紹我自己

—與網頁的相遇

我的名字是陳巧萱（CiaoSyuan,Chen）於106年（2017年）06月畢業於國立臺中科技大學的多媒體設計系。 大三在網頁公司實習時開啟了我對網頁製作的興趣與熱忱， 寫網頁就像是木雕工，需要靠經驗和時間的累積， 一點一滴地雕琢，而在研究技術與實踐技術的過程中讓我非常有成就感，因而踏上了這條永遠學不完的不歸路。

—關於我的專業與經歷

在業界有約1年的網頁開發的製作經驗， 擅長切版並熱愛以CSS3製作特效。 熟悉HTML、CSS、Javascript和JQuery語法之應用知悉如何修改WordPress模板並依照其需求修改樣式達成所需外貌， 有多樣作品皆以WordPress為基準製作。 出社會約1年左右至現今累積有13項的作品，在網頁前端開發這個領域不斷的學習的新技術提升自己的能力。

—關於我的興趣

除了寫網頁以外，我最喜歡的事情就是看電影和玩線上遊戲了，看點電影可以快速了解一個國家的文化特色和語言， 玩遊戲是最好放鬆心情的休閒活動。出遊時，比起拍人我更喜歡 拍風景並將這些拍下來的照片後製成明信片在過節時送給自己的親朋好友聯絡情誼。

HTML / HTML5

熟練度：★★★★★

用於編寫網頁架構和元素與插入媒體（Ex：影片和圖片等）。

CSS / CSS3

熟練度：★★★★★

Cascading Style Sheets（層疊樣式表），可替HTML添加樣式（顏色、大小等）。

RWD（Responsive Web Design）

熟練度：★★★★☆

響應式網頁設計，可讓網頁式應於各個解析度的設備（Ex：電腦和手機）。

jQuery

熟練度：★★★★☆

JavaScript函式庫，簡化HTML與JavaScript之間的操作，在製作網站時更快速方便。

JavaScript

熟練度：★★★☆☆

一種高階式語言，物件導向的直譯語言，可以撰寫不同的功能增加更多與使用者的互動性。

PhotoShop

熟練度：★★★★☆

影像處理軟體，點陣圖，可用於製作Banner等圖片，切版時經常使用。

Illustrator

熟練度：★★★☆☆

影像處理軟體，向量圖檔，可用來設計網頁版型和圖示。

Bootstrap

熟練度：★★★☆☆

CSS的框架，有寫好的功能和樣式只需要套用其類別即可使用。

Wordpress

熟練度：★★★★☆

豐富的插件和模板，擁有文章編輯的系統，可以與前端設計搭配製作出擁有後台的網站。
