About

介紹我自己

— 關於我的經驗

在業界有約 2 年的網頁開發經驗，製作網站約有 3 年的經驗， 擅長 RWD 切版和網頁動態效果。 使用 HTML、CSS 和 JQuery 完成使用者互動的介面。目前製作了1個平台網站、2個資訊網站、8個形象網站、1個活動網站與其他多項網站。在網頁前端開發這個領域不斷的學習的新技術提升自己的能力。

— 我與網頁的相遇

大三在網頁公司實習時開啟了我對網頁製作的興趣與熱忱，寫網頁需要靠經驗和時間的累積， 一點一滴地雕琢，而在研究技術與實踐技術的過程中讓我非常有成就感，因此開啟我的網頁旅程。

— 我對網站的想法

網頁是最快連結網路世界的平台，它呈現了許多資訊也最快傳達資訊，隨著時間推演技術的進步，網站已經能夠支援不同設備有不同的樣貌，如此多變的特性令我著迷，我熱愛寫網頁，它可以是一個溝通的橋樑也可以是一個樣貌的展示，一個好的網站，需要有很多技術的配合，精雕細琢後才能完美地呈現給使用者。

HTML / HTML5

熟練度：★★★★★

用於編寫網頁架構和元素與插入媒體（Ex：影片和圖片等）。

CSS / CSS3

熟練度：★★★★★

Cascading Style Sheets（層疊樣式表），可替HTML添加樣式（顏色、大小等）。

RWD（Responsive Web Design）

熟練度：★★★★☆

響應式網頁設計，可讓網頁式應於各個解析度的設備（Ex：電腦和手機）。

jQuery

熟練度：★★★★☆

JavaScript 函式庫，簡化 HTML 與 JavaScript 之間的操作，在製作網站時更快速方便。

JavaScript

熟練度：★★★☆☆

一種高階式語言，物件導向的直譯語言，可以撰寫不同的功能增加更多與使用者的互動性。

PhotoShop

熟練度：★★★★☆

影像處理軟體，點陣圖，可用於製作Banner等圖片，切版時經常使用。

Illustrator

熟練度：★★★☆☆

影像處理軟體，向量圖檔，可用來設計網頁版型和圖示。

Bootstrap

熟練度：★★★☆☆

CSS的框架，有寫好的功能和樣式只需要套用其類別即可使用。

Wordpress

熟練度：★★★★☆

豐富的插件和模板，擁有文章編輯的系統，可以與前端設計搭配製作出擁有後台的網站。
